# 五子棋 / Gomoku

## 游戏地址

- **本地双人对战**: https://yyuan168.github.io/gomoku/
- **联机对战**: https://yyuan168.github.io/gomoku/online.html

## 游戏说明

### 本地对战
在同一台设备上与朋友轮流落子。

### 联机对战
1. 打开联机版页面
2. 创建房间，获得 6 位房间号
3. 把房间号发给朋友
4. 朋友输入房间号加入
5. 开始对战！

## 技术栈

- 前端：原生 HTML5 + Canvas + WebSocket
- 后端：Node.js + ws (WebSocket)
- 部署：GitHub Pages + Render
